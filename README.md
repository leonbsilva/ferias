# Férias do Servidor

```
Aplicação de gestão de férias dos servidores.
```
### Setup
```
- Instalar os pacotes:
    - pip install -r requirements.txt
    
- Executar:
    - python manage.py migrate
    - python manage.py carregar_dados_iniciais
    - python manage.py createsuperuser
    - python manage.py runserver
```
